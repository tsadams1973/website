Using custom middleware to handle MsalUiRequired exceptions.

Posted on Aug 5, 2019

If you are using Microsoft Azure AD or Azure AD B2C as part of your IAM solution, odds are you are also using the Microsoft Authentication Library, MSAL.NET, to help you manage your tokens. This convenient library helps you acquire tokens that you can use to access APIs protected by Azure AD. It is useful and will save you a lot of time, but is not without its limitations.
